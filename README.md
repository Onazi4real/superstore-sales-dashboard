📊 Superstore Sales DashboardProject Overview

This project is an interactive Superstore Sales Dashboard built using Python, Pandas, Plotly, and Streamlit. It analyzes sales performance across different regions, categories, segments, products, and time periods to provide meaningful business insights.

Features

Interactive dashboard filters

KPI cards

Sales and Profit analysis

Monthly, Quarterly, and Yearly trends

Top Products analysis

Top States analysis

Sub-Category analysis

Business Insights

Dataset Statistics

Download filtered dataset

Responsive Streamlit interface
